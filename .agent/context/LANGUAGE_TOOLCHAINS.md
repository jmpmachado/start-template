# Language Toolchains — start-project

> Reference guide for compilers, interpreters, and package managers supported by this template.

---

## 1. Supported Toolchains

| Language | SDK / Compiler | Package Manager | CI Commands |
|---|---|---|---|
| **TypeScript / JS** | Node.js ≥ 24 | npm ≥ 10 | `npm run lint && npm test` |
| **.NET Core** | .NET 9 SDK | NuGet | `dotnet build && dotnet test` |
